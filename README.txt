Requirements: You need two players
What it is: Connect 4 Game
What does it do: Replicate the Connect 4 Game 
How does it work: 
* User x goes first and selects a column to drop the x 
* User o goes second and selects a column to drop the 0
* First player to get 4 In a row, column, or diagonal, then the player will win
